# Road map building

## Requirements

- Exists a Team
- Exists a Users
- Exists a Sprints
- Exists a Topic
- Exists one or more owner of Team
- A Team can grouper a or more users
- A owner can be invite more users for own Team
- A topic is any generic for vote
- Users can be votes in topics
